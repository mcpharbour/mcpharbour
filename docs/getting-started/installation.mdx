## One-line Install

The install script downloads pre-built binaries, registers the daemon as a system service, and starts it. No Python or package manager required.

<CodeGroup>

```bash Linux / macOS
curl -fsSL https://mcpharbour.ai/install.sh | bash
```

```powershell Windows
irm https://mcpharbour.ai/install.ps1 | iex
```

</CodeGroup>

This installs two commands:

| Command | Purpose |
|---------|---------|
| `harbour` | Server administration — manage servers, identities, policies, daemon |
| `harbour-bridge` | Agent bridge — lightweight, no admin access |

## What the Install Script Does

1. Downloads the latest release binaries for your platform
2. Installs `harbour` and `harbour-bridge` to `~/.local/bin` (Linux/macOS) or `%LOCALAPPDATA%\mcp-harbour\bin` (Windows)
3. Registers the daemon as a system service:
   - **Linux**: systemd user service
   - **macOS**: launchd user agent
   - **Windows**: Task Scheduler task
4. Starts the daemon on `127.0.0.1:4767`

The daemon starts on boot and restarts on crash.

You can also download binaries manually from the [GitHub Releases](https://github.com/mcpharbour/mcpharbour/releases) page.

## Verify

```bash
harbour status
```

## Development Setup

For contributing or local development:

```bash
git clone https://github.com/mcpharbour/mcpharbour.git
cd mcp-harbour
uv venv
source .venv/bin/activate
uv pip install -e ".[dev]"
```

This installs in editable mode — changes to the source take effect immediately.

```bash
# Run tests
pytest

# Run the daemon locally
harbour serve
```
